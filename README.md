# LiShihhsing Personal Site

Personal website built with Astro, TypeScript, and Tailwind CSS.

## Local Development

PowerShell may block `npm.ps1`. If that happens, use `npm.cmd`.

```powershell
npm.cmd install
npm.cmd run dev
```

Open:

```text
http://localhost:4321
```

## Content Files

- `src/data/profile.ts`: profile and social links
- `src/data/projects.ts`: project cards
- `src/data/education.ts`: education and course progress
- `src/data/plans.ts`: public roadmap
